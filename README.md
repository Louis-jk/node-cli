# Merkle-Tree JSON file Maker

### Install packages

```bash
npm install
```

### Build

```bash
npm run build
```

### If you want to global install

```bash
npm install -g
```

### Usage

```bash
merklemanager
```

![Alt text](image-5.png)

```bash
merklemanager -t <Your excel file path>
```

then 「tree.json」file and 「treeRoot.txt」 file are created.

![Alt text](image-6.png)
